
*** How to run the application. ***


1.  Open the file containing the code, once opened in Visual Studio there will be a play button next to
	the file name. Click that button to build and start the application. 

2.  Once the application has opened, Click on the button labled continue to use the application further.

3.  Enter All the infomation the application askes for to avoid errors, onces filled in Click Continue. 

4.  After clicking the continue button you, the user will be displayed with a choice if you would like 
	to enter another module or not. IF selected yes the choice will disapear and you will be able to 
	enter more modules. IF selected no the application will proceed.

5.  The application will display the modules that you, the user have entered with the amount of self study 
	hours required per week for each module. You can click Next when you are ready for the application to 
	persist.

6.  Provide all the infromation required by the application to ensure there are no errors. Onces you, the 
	user has entered the 3 fields of infromation by clicking display the application will the calculate how 
	much time left you have to self study in that week. This will be displayed in the list box bellow the button

7.  Once you have entered all your modules and see how much more time you have left, you may click the exit 
	button to exit the application.
